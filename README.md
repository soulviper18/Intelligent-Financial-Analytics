# Intelligent Financial Analytics Platform (Athena)

A production-grade backend system built using Spring Boot, JWT-based authentication,
role-based authorization, audit logging, and PostgreSQL.

## Features
- JWT Authentication (Access & Refresh Tokens)
- Role-Based Authorization (USER / ADMIN)
- Secure Password Management (BCrypt)
- Audit Logging for Security Events
- Request Tracking with Filters
- Environment-based Configuration (Dev / Prod)
- PostgreSQL Persistence
- Global Exception Handling

## Tech Stack
- Java 17
- Spring Boot 4
- Spring Security
- JWT (JJWT)
- PostgreSQL
- Hibernate / JPA
- Gradle
- Swagger / OpenAPI

## Security Highlights
- Stateless authentication
- Token blacklisting on logout
- Method-level authorization
- Secure configuration using profiles
